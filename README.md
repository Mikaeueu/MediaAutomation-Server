# 🎛️ ChurchFlow

Sistema de automação de mídia para igrejas, com foco no controle de
lives, integração com OBS, e gerenciamento remoto via celular.

------------------------------------------------------------------------

## 🚀 Funcionalidades

-   🔐 Sistema de login simples
-   📺 Controle do OBS via WebSocket
-   🎥 Geração automática de lives (YouTube / Facebook)
-   🔑 Retorno automático de chaves de transmissão
-   🎼 Integração com Holyrics para troca automática de cenas
-   ⏱️ Retorno automático para cena padrão após inatividade
-   📴 Agendamento de desligamento do computador
-   📱 Painel web responsivo (controle via celular)
-   ⚙️ Inicialização automática com o sistema

------------------------------------------------------------------------

## 🧠 Tecnologias utilizadas

-   Python
-   Flask
-   OBS WebSocket
-   HTML / CSS / JavaScript
-   Bootstrap
